# Coctels OPS | ¡¡ QUE CHIMBA UN OPS !!

Este es el prototipo profesional de **Coctels OPS**.

## 🚀 Cómo obtener tu enlace para compartir:
Para que tus clientes en Villa Hermosa puedan pedir sus granizados desde la web, sigue estos pasos:

1. Ve a la [Consola de Firebase](https://console.firebase.google.com/).
2. Selecciona tu proyecto actual.
3. En el menú de la izquierda, entra en **Build > App Hosting**.
4. Haz clic en "Get Started" y sigue el asistente para conectar tu código.
5. Al finalizar, Firebase te entregará una URL pública (ej: `https://tu-proyecto.web.app`).

## 📱 Características optimizadas:
