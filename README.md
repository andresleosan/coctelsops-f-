
# Coctels OPS | ¡¡ QUE CHIMBA UN OPS !!

Este es el prototipo profesional de **Coctels OPS**.

## 🚀 Cómo obtener tu enlace para compartir:
Para que tus clientes en Villa Hermosa puedan pedir sus granizados desde la web, sigue estos pasos:

1. Ve a la [Consola de Firebase](https://console.firebase.google.com/).
2. Selecciona tu proyecto actual.
3. En el menú de la izquierda, entra en **Build > App Hosting**.
4. Haz clic en "Get Started" y sigue el asistente para conectar tu código desde GitHub.
5. Al finalizar, Firebase te entregará una URL pública (ej: `https://tu-proyecto.web.app`).

## 💻 Conectar con VS Code:
Si quieres abrir este código en tu computadora local:
1. Asegúrate de tener instalado [Node.js](https://nodejs.org/).
2. Clona tu repositorio de GitHub: `git clone https://github.com/andresleosan/Coctelsops-F-.git`.
3. Entra a la carpeta y ejecuta `npm install` para instalar todas las dependencias.
4. Ejecuta `npm run dev` para ver el proyecto en `localhost:3000`.

## 📱 Características optimizadas:
- **Diseño Mobile-First:** Todo está pensado para que el cliente pida desde su celular.
- **Panel OPS Admin:** Gestiona tus pedidos en tiempo real desde `/admin/dashboard`.
- **Estética Neón:** Colores Magenta y Cian sobre fondo negro para resaltar tu marca.
