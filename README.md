
# Coctels OPS | ¡¡ QUE CHIMBA UN OPS !!

Este es el proyecto profesional de **Coctels OPS**, optimizado para dispositivos móviles y con integración de Firebase.

## 🚀 Cómo subir este código a tu GitHub:
Para que tu código aparezca en `https://github.com/andresleosan/Coctelsops-F-`, sigue estos pasos en la **Terminal**:

1. **Abrir la terminal:** En VS Code ve a `Terminal > New Terminal`.
2. **Inicializar Git:** 
   ```bash
   git init
   ```
3. **Vincular tu repositorio:**
   ```bash
   git remote add origin https://github.com/andresleosan/Coctelsops-F-.git
   ```
4. **Preparar archivos:**
   ```bash
   git add .
   ```
5. **Crear el primer commit:**
   ```bash
   git commit -m "Versión final con Panel OPS y Firebase"
   ```
6. **Subir el código:**
   ```bash
   git push -u origin main
   ```
   *(Si te pide autenticar, sigue los pasos en pantalla para iniciar sesión en GitHub)*.

## 📱 Características:
- **Diseño Mobile-First:** Todo está pensado para que el cliente pida desde su celular.
- **Panel OPS Admin:** Gestiona tus pedidos en tiempo real desde `/admin/dashboard`.
- **Estética Neón:** Colores Magenta y Cian sobre fondo negro.
- **WhatsApp Directo:** Integración total con el número 324 554 5530.

## 💻 Desarrollo Local:
1. Instala las dependencias: `npm install`
2. Corre el proyecto: `npm run dev`
3. Abre `http://localhost:3000` en tu navegador.
